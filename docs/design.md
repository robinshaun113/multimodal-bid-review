# P3 · 多模态 IDC 应标书智能审核 —— 需求设计文档（Day36）

> 诚信原则：场景为行业公认痛点 + 自驱实践项目（非谎称亲历）；数据为真实应标书，本地开发、演示脱敏；指标标注样本量、不作虚荣卖点。

---

## 1. 定位与场景

数据中心（IDC）应标技术标书通常含拓扑图、机柜布局图、SLA 条款、技术规格偏离表、报价单等**图文混合的异构信息**，人工审核耗时、易漏关键项（如 SLA 数值不达标、强制项未响应）。

本项目自驱构建一个**多模态审核助手**：读入一份应标书（docx），自动核对其技术方案是否达标、对招标要求的响应是否完整、有无遗漏或不利条款，输出结构化审核报告。核心新能力是**多模态**——能"读懂"标书里的图（前两个项目 P1-RAG / P2-Agent 未涉及）。

## 2. 目标用户与价值

- **用户**：招标方评标人员 / 投标方自查人员。
- **价值**：把"人工逐页翻几百页图文标书"压缩为"AI 逐项核对 + 定位原文出处"，降低漏检、提升审核一致性。

## 3. Pipeline（五环）

```
①docx解析 → ②VLM读图 → ③多模态RAG入库 → ④checklist审核 → ⑤生成报告
```

| 环节 | 做什么 | 状态 |
|------|--------|------|
| ① docx 解析 | docx(zip) 拆出文字块(document.xml) + 图片(word/media) | ✅ src/docx_parser.py |
| ② VLM 读图 | 每张图经 base64 → qwen-vl-max → 文字描述 | ✅ src/vlm.py |
| ③ 多模态 RAG 入库 | 文字块直接入向量库；图的"文字描述"入库 + metadata 存原图路径；检索同时带回文本与原图 | ⏳ Day39 |
| ④ checklist 审核 | 按审核清单逐项去 RAG 检索、判断达标性，给出结论+出处 | ⏳ Day40 |
| ⑤ 生成报告 | 汇总各项结果为结构化报告（达标/缺失/风险 + 原文引用） | ⏳ Day43 |

**多模态灵魂（③）**：图片无法直接做向量语义检索，故"先用 VLM 把图转成文字描述，再让描述入库"，并用 metadata 记住描述对应哪张原图——这样"机柜布局是否合理"这类问题也能命中相关图。

## 4. 统一数据格式

解析层输出统一结构，贯穿全 pipeline：
```
{
  "text_chunks":  [{"text": "...", "kind": "paragraph/table"}],
  "image_chunks": [{"image_path": "...", "rid": "imageN.jpeg", "desc": "<VLM描述>"}]
}
```

## 5. 技术选型

- 解析：python-docx（文字/表格）+ zipfile（取内嵌图）
- VLM：qwen-vl-max（百炼 OpenAI 兼容接口）
- 向量库/RAG：复用 P1 经验（Chroma + embedding）
- 报告：Jinja2 模板 → Markdown（Day43）
- 前端：Gradio（Day44）

## 6. 审核 Checklist（核心业务逻辑）

设计原则：**业务分量（废标红线）+ 技术展示分量（多模态特色）兼顾**。
以下项将落成 `checklist.yaml`（Day40），审核 Agent 逐项去 RAG 检索判断。

| # | 审核维度 | 具体检查点 | 类型 |
|---|----------|-----------|------|
| 1 | **强制响应项** | 招标 ▲/★ 必答项是否逐条响应，有无遗漏或负偏离 | 废标红线 |
| 2 | **关键技术指标达标** | 机房等级（国标 A 级 / T3+）、SLA 可用性（≥99.9%?）、机柜功率密度、PUE | 废标红线 |
| 4 | **运维服务承诺** | 故障响应时限、7×24 值守、到场时间 | 废标红线 |
| 5 | **图纸完整性**（多模态特色）| 拓扑图 / 机柜布局图**是否提供、是否清晰、内容是否合理**（VLM 读图判断）| 技术展示 |
| 6 | **商务 / 风险** | 报价合理性、违约条款、有无不利表述 | 废标红线 |

> 说明：第 5 项是 P3 "多模态"能力的落点——评委关心的图纸质量，靠 VLM 读图审核，是本项目区别于纯文本 RAG 的核心。第 3 类（电力/制冷承诺）弱化为次要项（信息多与第 2 项交叉）。

> 每项审核输出：`{项: ..., 结论: 达标/缺失/风险, 依据: RAG检索到的原文/图, 出处: 页码或图ID}`。
